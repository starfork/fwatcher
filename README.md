## Description
Watch file change and send notify to your telegram

## Config Sample 
you should put `config.yaml` in the running path
```yaml
token: "your token"
chatid: yourchatid
max: max length of the send
interval: 10
path:
  - "path1"
  - "path2"
  - "path3"
```

## install

```
go install https://github.com/starfork/fwatcher@latest
```

## Usage
```
fwatcher 
```
